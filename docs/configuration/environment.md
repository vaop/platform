# Environment Variables

These settings are configured during installation via the setup wizard. You can also edit them directly in the `.env` file.

## Application

| Variable | Description |
|----------|-------------|
| `APP_NAME` | Your VA name, shown in the browser title and emails |
| `APP_URL` | The full URL to your site (e.g., `https://crew.yourva.com`) |

## Database

| Variable | Description |
|----------|-------------|
| `DB_HOST` | Database server (usually `localhost` or `127.0.0.1`) |
| `DB_DATABASE` | Database name |
| `DB_USERNAME` | Database username |
| `DB_PASSWORD` | Database password |
